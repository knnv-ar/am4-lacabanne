## Envío de datos de Arduino a Resolume Archivo Editar título 51.3KB

Ejemplo de comunicación entre Arduino, Processing y Resolume a través de puertos Serie y OSC.

Arduino envía sus datos a la computadora por el puerto Serie. Pero Resolume esta preparado para recibir datos del teclado de  la computadora, del puerto MIDI ó de un puerto OSC (no del puerto Serie). Por esta razón, utilizamos Processing para recibir los datos del puerto Serie y reenviarlos a un puerto OSC, para que puedan ser recibidos por Resolume.

**Arduino** < SERIE > **Processing** < OSC > **Resolume**
