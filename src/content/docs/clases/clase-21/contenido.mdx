## Envío de datos entre Arduino, Processing y Pure Data Archivo 93.1KB `.pdf`

Ejemplo de comunicación entre Arduino, Processing y Pure Data a través de puertos Serie y OSC.

Arduino puede conectarse a través del puerto serie con un sólo programa a la vez (Processing o PD, por ejemplo). Si para un proyecto necesitamos comunicar los 3, sumamos a la cadena el 3er programa a través de OSC. Recomendamos conectarlos del siguiente modo:

**Arduino** < SERIE > **Processing** < OSC > **PD**

Ver "Arduino - Comunicacion Serie.pdf" (página 14).

NOTA: Este ejemplo requiere de algunas librerias de PD. En caso de utilizar PD-extended o PD-L2Ork, ya vienen instaladas. En caso de utilizar PD-vanilla, se deberán instalar manualmente.
